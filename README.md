# RoSHAMbrawl (name subject to change)

RoSHAMbrawl is a simiplistic pokemon autobattler that uses generation 1 pokemon and only fire, water, and grass types. Thus the name.

The users will create teams of 3 pokemon and export them as codes. The admin, the gymleader will then take these codes and pit the 2 teams against eachother until one of the team wins.
## Features
- user login(trainer/gymleader)
- team creation and editing
- pokemon selection
- export and import team codes
- automatic battle system
## Tech stack 
- Java
- JavaFX
- SQLite database
- Gradle
## How to run 
- clone the repo
- open in intelliJ
- run using gradle (gradlew run)
